Create an “OK to Wake Clock” Android app. The app should allow the user to set a wake time, a wake alarm duration (in minutes), a brightness value (from 0 to 10), and an optional pre-wake duration (in minutes). The app should display the current time in large white font on a black background. This is the default screen. The brightness of the default screen should be determined by the brightness value set by the user. At the wake time the background should change to green and stay green for the duration set by the “wake alarm duration”. If “pre-wake duration” is set the background should turn orange X minutes prior to the wake time, where X is the “pre-wake duration”. The screen should not go to sleep while the app is in the foreground. The app should open to the default screen. There should be a systems setting wheel icon in the bottom right corner of the default screen that takes the user to the settings menu. Make sure there’s a back button in the settings menu that takes the user to the default screen. Assume that I have no prior experience developing for Android OS. Add detailed instructions for how to create and deploy this app.
